## MacroTracker

MacroTracker is a basic tracking tool implemented in PowerShell. Why? Because I can.
Also, I tend to find the existing apps overly complicated and usually just keep track in a calculator or terminal anyways, so I'll find it mildly helpful. It'd be cool if someone else did too.

This *should* work on anything that will run PowerShell. 

Maybe one day I'll feel like writing a phone app for it.
